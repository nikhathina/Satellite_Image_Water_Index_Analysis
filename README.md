# Satellite Image Water Index Analysis

## Project Overview
Python script for calculating Normalized Difference Water Index (NDWI) from Sentinel-2 satellite imagery to identify water bodies and monitor hydrological features.

## Technical Skills Demonstrated
- **Remote Sensing**: NDWI calculation, satellite image processing
- **Geospatial Analysis**: Rasterio, coordinate reference systems
- **Data Visualization**: Matplotlib, custom colormaps
- **Python**: NumPy, raster data handling

## Methodology
1. Load Sentinel-2 Green and NIR bands
2. Calculate NDWI using standard formula
3. Handle no-data values and data cleaning
4. Generate visualizations and analysis
